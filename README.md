# HustleHub (The Side Hustle Marketplace)

HustleHub is a community-driven marketplace built for the modern gig economy. Whether you need an errand run or you're looking to turn your skills into a side hustle, HustleHub connects people who have tasks with those looking to earn extra income.

## Key Features

- Secure Authentication: Robust user login and registration powered by Firebase.
- Side Hustle Dashboard: A clean interface for managing your profile and browsing opportunities.
- Skill Showcasing: Professional user profiles to highlight skills (tutoring, labor, tech, etc.).
- Data Protection: Advanced route protection to ensure user privacy and security.

## Tech Stack

- Framework: Next.js (App Router)
- Styling: Tailwind CSS
- Database/Auth: Firebase & Cloud Firestore
- Deployment: Vercel

## Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`
3. Run locally: `npm run dev`
